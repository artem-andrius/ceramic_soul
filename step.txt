29) В цьому практичному завданні ми вже будемо користуватись rem.
Rem відталкується від font-size який стоїть в html, за замовчуванням там 16px (наприклад якщо потрібно 32px то це буде 2rem) (32/16=2)

Найкраще користуватись функцією яку я підключаю @use '/src/sass/pxtorem.scss';

    @use "sass:math";

    @function rem($pixel) {
      @if math.is-unitless($pixel) {
        @return math.div($pixel, 16) + rem;
      } @else {
        @error 'Do not use units of measurement when working with the rem() function. Pass only numbers as arguments.';
      }
    }

тоді я задаю px в rem за допомогою font-size: pxtorem.rem(32);

але краще

@use '/src/sass/pxtorem.scss' as *;
тоді ж буде font-size: rem(32);

Але самий зручний і самий ленивий варіант яким буду користуватись це https://www.npmjs.com/package/postcss-pxtorem
(Не забуваю вказати цю команду, якщо перезагружав пк Set-ExecutionPolicy RemoteSigned)
вводимо команду npm i postcss-pxtorem (взяв з сайту)

Створюю файл postcss.config.js в корні і змістом

    module.exports = {
      plugins: {
        "postcss-pxtorem": {
          rootValue: 16,
          propList: ["*"], // Преобразовывать все свойства
          selectorBlackList: [], // Селекторы, которые нужно исключить из преобразования
          replace: true, // Заменять px на rem
          mediaQuery: true, // Преобразовывать px внутри media queries
          minPixelValue: 0, // Минимальное значение px для преобразования
        },
      },
    };

І потім px автоматично стають rem
